# IPL Analytics Dashboard

## Overview
This Power BI dashboard analyzes IPL match data and provides insights into team performance, venues, toss decisions, and player statistics.

## Tools Used
- Power BI
- DAX
- Excel

## Key KPIs
- Total Matches
- Total Teams
- Total Venues
- Total Runs
- Total Wickets

## Dashboard Features
- Team Performance Analysis
- Venue Analysis
- Toss Decision Impact
- Player of the Match Analysis
